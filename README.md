# RaceToolPro
【WIP】小米赛道大师后处理工具

v1 支持切分帧后，逐帧获取ocr数据，并保存到表格。 
使用 crop 方式裁剪所有数据的对应区域，再用置信度要求低的方法 ocr，最后选置信度最高的一个，以解决“查不到 0” 的问题。 
v2 按照每秒 3 帧绘图。发现小米的赛道视频的驾驶数据实际只支持 3~10Hz 的刷新率

从单圈赛道视频，生成赛道热力图
蓝色代表加速，红色代表减速；红蓝渐变代表半刹，半油
标记弯速。直接根据速度的“谷值”选点。从而避免“识别弯道”或“手动配置弯道”。
反转颜色变成白底黑线。
对于 ocr 识别，先裁切图片，再识别。
对视频逐帧识别比较慢，可以考虑把fps调到2-10先跑一个preview，再给出完整fps的结果
侧向g值需要旋转图片后识别

目标检测+局部抠图，找到箭头的坐标，将整个箭头的凸包（矩形）裁切下来，覆盖在标准图上。这个方案会比按相加的方案更稳定。标准图通过保存第一个箭头的完整图片得到。
弯速使用相同大小的字体写在箭头的左侧，防止挡住赛道线。
白色底色/偏纸张颜色的底色的赛道图，会更好调颜色一些。
